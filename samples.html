Chat Galaxy

An application for online chat. You need to choose a name an avator before entring the main chat room. Later you can switch rooms. Main purpose of this application was to practice my socket.io and event and emit skills by using express and nodejs in back-end alongside JQuery, html and javascript on front-end. Deployed and functional on Heroku.

Dev Connector

Originally was a course by Traversy Media on udemy but later I changed some aspects and customized it. Vast technologies were used in this project. It was a MERN stack project, starting with mongo DB as NoSql database, NodeJs and express on server, and ReactJs on front-end side of the project. Also it included various libraries such as JWT for authentication and authorization, express validator for REST value checks and axios. Finally deployed on Heroku.
